About the website. In the middle of the screen are nodes, in nodes center are the bias. If the weight is positive it will be red, if it is negative it will be blue. If the weight is more red it means it is bigger then other positive weights, if it is more blue it means it is more negative then other negative weights. The table shows input, expected output and predictions, for Iris dataset you can see each flowers probability. Above are are 3 buttons, pressing them you will equip one of the written datasets. You can add or remove layers and nodes in each layer, enable ReLU or Sigmoid, change Learning rate, number of iterations and if Iris dataset is selected select if to use Softmax. Below Backpropagations is current cost of the network. Also there is a linear graph that shows progress of a network.
